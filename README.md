### Prerequisites

### Installation

1. Get a free API Key at (https://newsapi.org/)

2. Clone the repo

```sh
git clone https://github.com/markoeltiger/OTNews.git
```


3. Enter your API in `Constats.kt`

```JS
const API_KEY = 'ENTER YOUR API';
```
